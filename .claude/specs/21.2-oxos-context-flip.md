# Spec: Context Section with Card Flip

## Roadmap description
Interactive Context section on Oxos Platform page with a flippable Supabase card showing database tables.

## Overview
This release adds the Context section to the Oxos Platform page, featuring an interactive Supabase card that flips on click to reveal database table information. The card uses CSS transitions for smooth animation and is designed to be extensible for future data systems like Snowflake, PostgreSQL, APIs, and S3. This completes the Oxos Platform dashboard with all three sections: Business Outcomes, Context, and Learnings.

## UI/UX notes
- Card flip animation using CSS transform and transition
- Front shows: Database icon (lucide: database), "Supabase", "Primary Data System" 
- Back shows: List of database tables (initially hardcoded)
- Click/tap toggles the flip state
- Responsive: flip card works on mobile and desktop
- Extensible design: additional data system cards can follow same pattern

## Rules for implementation
- No SQLAlchemy or ORMs — use raw sqlite3/psycopg2 queries only
- Parameterised queries only
- Passwords hashed with werkzeug — `generate_password_hash` / `check_password_hash`
- Use CSS variables — never hardcode hex values
- All templates extend `base.html`
- Auth guard on protected route — check `session.get("user_id")` and redirect to `/login` if missing
- Currency displayed in ₹ — format amounts as `₹{amount:,.2f}`
- Dates stored as `YYYY-MM-DD` strings — validate with `datetime.strptime(value, "%Y-%m-%d")`
