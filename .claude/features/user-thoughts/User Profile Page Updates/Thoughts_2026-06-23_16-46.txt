User Profile Page Updates

This enhancement should focus on consolidating key functionality into the Profile page and simplifying the overall navigation experience.

1. Change Password Functionality
- Change Password button exists but is non-functional
- Enable the feature on the Profile page, allowing users to securely update their passwords without leaving the profile experience

2. Quick Add Expense from Profile
- Replace the existing Add Expense navigation item with a prominent "+" button positioned to the right of the Profile title heading (horizontally aligned, right side of the page)
- Clicking this button should open a modal containing the same fields and functionality currently available on the Add Expense page
- Upon submission, the expense should be saved directly to the user's expenses list
- Once this functionality has been successfully migrated to the Profile page, remove the Add Expense nav-bar and delete any associated pages, routes, components, and unused code
- It would also be valuable to explore additional ideas for making expense entry faster and more intuitive, such as quick-add templates, recurring expenses, or smart category suggestions

3. Embedded Analytics Dashboard
- Add a compact Analytics section between the profile card and filter bar
- This section should provide visual insights through charts, graphs (maybe line graph) that displays spending trends over time
- Users should be able to switch between multiple views within the same analytics container (e.g., spending trends, category breakdowns, monthly comparisons, etc.) without navigating to a separate page
- The section should remain concise and visually clean while still providing meaningful insights
- Additional ideas for analytics visualizations and interactions would be valuable to explore to maximize value without overwhelming the page
- After the analytics functionality has been fully integrated into the Profile page, remove the standalone Analytics navbar and delete any associated pages, routes, components, and obsolete code

The end goal is to make the Profile page the central hub for account management, expense creation, and personal financial insights while reducing navigation complexity and improving the overall user experience.
